# **App Name**: CodeFlow

## Core Features:

- User Authentication: Allow users to create accounts, log in, and manage their profiles.
- Coding Profile Connection: Enable users to connect their coding profiles from platforms like Codeforces and LeetCode.
- Activity Feed: Display a real-time activity feed of coding problem solutions from connected users.
- Messaging Feature: Provide a messaging system for users to chat, discuss problems, and collaborate.
- Profile Page: Allow each user to see and customize their own profile page, which includes their recent activities.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey knowledge and focus.
- Background color: Very light lavender (#F1F3FF).
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements and updates.
- Body font: 'Inter', sans-serif, for a clean and modern look.
- Headline font: 'Space Grotesk', sans-serif, providing a computerized feel.
- Use clear and consistent icons for navigation and actions, such as problem submissions, chat, and profile settings.
- A feed-focused layout, inspired by LinkedIn, and the addition of dedicated message boards similar to Discord.
